Flexbox

You have "flex containers" and "flex items" inside them

flex-direction sets the main axis/layout axis of the 1D styling

  • flex-pack

justify-content

This moves where flex items are positioned along the main axis. When flex-direction = row the axis is horizontal, and vice versa for column.

"start" of the axis is top/left, unless you use flex-direction row-reverse or column-reverse

flex

flex property of flex items sets their width as parts of 100% width

flex-item with flex: 2

flex-item with flex: 15

flex-item flex: 3

Here there are a total of 15+2+3 parts = 20

If there are 20 parts of the width available then 1 part = 5% of the width. If you change the total values of flex in a flex container then the width/part calculation will change!

flex-wrap (and flex-flow)

flex containers don't wrap their items by default. Can be set to no-wrap, wrap, wrap-reverse.

flex-flow is a shorthand that combines flex-direction and flex-wrap

flex-flow: <flex-direction> <flex-wrap>
