Flexbox

You have "flex containers" and "flex items" inside them

flex-direction sets the main axis/layout axis of the 1D styling

Outdated stuff mentioned in the book that doesn't work anymore:

  • display: flexbox;
  • flex-pack
    • This seems to be the old version of justify-content
  • flex-order
    • This seems to be the old version of order

justify-content

This moves where flex items are positioned along the main axis. When flex-direction = row the axis is horizontal, and vice versa for column.

"start" of the axis is top/left, unless you use flex-direction row-reverse or column-reverse

flex-direction and order

Row (default), column, row-reverse, column-reverse

"start" of the axis is top/left, unless you use flex-direction row-reverse or column-reverse. Below are row and row-reverse flex containers, both with justify-content set to flex-start

flex items can have a say about where they go in the order, using the order CSS property.

Order goes along the layout axis, from wherever flex-start is...:

Element 2 with order:1
