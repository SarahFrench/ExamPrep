# Define element, style, and attribute selectors

You can select by tag name eg div:

```css
div {
  color: red;
}
```

You can select by class or id:


```css
.card {
  color: blue;
}

#specificCard {
  color: grey;
}
```

and also by attributes:


```css
input[required] {
  border: red 2px solid;
}

[class$=important] {
  border: red 2px solid;
}
```
