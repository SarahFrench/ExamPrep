Important text!

  • $('[id=heading]') selects elements with id 'heading'
  • $('[class$=important]') selects elements with classes ending in 'important'
  • $('[class^=text]') selects elements with classes starting with 'text'
  • $('[name*=a]') selects elements with the name attribute set to a value including the letter "a"
  • $('[required]') selects elements with the required attribute
  • $('*') selects EVERYTHING
