Trying out XML parsing in command line

Click here to trigger a XHR request to a Joke API:

In the console you can play around with these variables:

  • jokeXMLObject : request.responseXML return value
  • jokeString : request.responseText return value
  • request : the XHR Object

Try parsing the string with $.parseXML(<string>)
